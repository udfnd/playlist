# Interview: 3D Playlist Visualization

## Round 1: Scope
Question: What is the scope of the 3D playlist visualization?
Answer: 3D album carousel with playback functionality. Albums displayed in a cylindrical container arranged in a circle, rotating in 3D. Clicking an album shows playlist details with a simple playback UI. Mouse/touch rotation control.

## Round 2: Constraints
Question: Where does the data come from?
Answer: Start with mock data (hardcoded album/playlist data), plan for YouTube Music API integration later as a separate SPEC.
